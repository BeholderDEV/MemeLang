Memelang - Tabelas de Compatibilidade



  • INT / INT = FLOAT
  • INT RELACIONAL, LOGICO INT = BOOLEAN
  • INT (+-*/) FLOAT = FLOAT
  • INT RELACIONAL, MOD, LOGICO FLOAT = BOOLEAN
  • FLOAT (+-*/) INT = FLOAT
  • FLOAT RELACIONAL, LOGICO INT = BOOLEAN
  • FLOAT RELACIONAL, LOGICO FLOAT = BOOLEAN
  • CHAR + CHAR = STRING
  • STRING + CHAR = STRING
  • CHAR + STRING = STRING
  • CHAR RELACIONAL, LOGICO STRING = BOOLEAN
  • STRING RELACIONAL, LOGICO CHAR = BOOLEAN
