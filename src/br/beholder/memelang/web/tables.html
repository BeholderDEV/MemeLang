Modelo de tabelas de compatibilidade

Exceções:

  • INT / INT = FLOAT
  • INT (+-*/) FLOAT = FLOAT
  • FLOAT (+-*/) INT = FLOAT
  • CHAR + CHAR = STRING
  • STRING + CHAR = STRING
  • CHAR + STRING = STRING

Compatibilidade de Operações

int char float string
int + - / * erro + - / * erro
char erro + erro +
float + - / * erro + - / * erro
string erro + erro +

Compatibilidade de Atribuições

int char float string
int ok erro aviso erro
char erro ok erro aviso
float ok erro ok erro
string erro ok erro ok
